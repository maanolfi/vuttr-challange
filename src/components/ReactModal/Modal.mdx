# Modal

Você deve importa o ReactModal e passar duas props obrigatorias, que são:
open e handleModal

<Playground>
  <ReactModal open={false} handleModal={() => {}} label="Name" >
    <span>Teste</span>
  </ReactModal>
</Playground>


A props open recebe o valor do campo check (true / false)

```js
  const [check, setCheck] = useState(false);
```
